## About Me

</div>

Full Stack Developer who ships complete, deployed systems — not side projects left at 80%. My BBA background in Accounting & Finance shaped how I think about software: business logic matters, data has consequences, and the system needs to work before it goes live.

I've integrated Stripe payment pipelines, Socket.IO real-time sync, JWT and Firebase auth, Cloudinary media management, and the Anthropic Claude API across production projects. I care about authorization at the API layer, conflict resolution in real-time systems, and code that a second developer can read without a tour.

Currently deepening expertise in backend architecture, system design, and scalable API patterns.

### DevFolio Analyzer
**AI-Powered · GitHub Profile Analysis**

[![Preview](./preview/devfolio-analyzer.png)](https://devfolio-analyzer.vercel.app/)

Fetches live GitHub data, scores profiles across six weighted dimensions using a deterministic model, then layers Anthropic Claude API insights on top. The scoring pipeline runs independently of the AI layer — graceful fallback ensures consistent output when the API is unavailable. Authenticated users get persistent history with score-over-time charting; guests get instant analysis with no friction.
